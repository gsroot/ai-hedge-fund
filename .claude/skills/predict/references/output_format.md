
열 설명:
- **총점**: 전략별 최종 점수 (0-10+)
- **펀더**: fundamental 점수
- **모멘**: enhanced_momentum 점수
- **앙상블**: 투자자 앙상블 가중 평균
- **수익률**: 1년 후 예상 수익률
- **P/E**: 주가수익비율 (없으면 `N/A`)
- **ROE**: 자기자본이익률 (없으면 `N/A`)
- **PEG**: Price/Earnings to Growth (없으면 `N/A`)
- **강세 투자자**: 점수 ≥ 7인 투자자 (최대 3명)
- **주요 요인**: 상위 2개 긍정 요인 (펀더멘털 전략만)

## 출력 규칙

1. **항상 Section 1~8 순서를 지킴** (해당 데이터가 없는 섹션은 생략)
2. **신호 표기 통일**: `🟢 강력매수` / `🔵 매수` / `⚪ 보유` / `🟡 약한매도` / `🔴 매도`
3. **숫자 포맷 통일**: 수익률 `+15.3%`/`-5.2%`, P/E `23.1`, ROE `28.0%`, PEG `1.5`, 시총 `$3,400B`/`₩320조`
4. **N/A 처리**: 데이터 없으면 `N/A` 표시, 빈칸으로 두지 않음
5. **구분선**: `═` (섹션 구분), `─` (테이블 헤더 아래)
6. **순위표 열 정렬**: 좌측 정렬 (종목명), 우측 정렬 (숫자), 좌측 정렬 (텍스트)
7. **display 초과 종목**: 순위표에 표시하지 않되, 요약 통계(Section 3~7)는 전체 분석 결과 기반
8. **한국 종목**: 시총은 `₩` 접두사 + 조/억 단위 (`₩320조`, `₩5,000억`), 종목명은 한글 표시
