## 기술적 분석

### 추세 분석 (Trend Following)

#### 이동평균 시스템
```
단기: 8일 EMA, 21일 EMA
중기: 50일 SMA
장기: 200일 SMA

황금 배열 (Bullish): 가격 > 50일 > 200일
데드 배열 (Bearish): 가격 < 50일 < 200일
```

#### 크로스오버 신호
```
골든크로스: 50일선이 200일선 상향 돌파 → Bullish
데드크로스: 50일선이 200일선 하향 돌파 → Bearish
```

### 모멘텀 분석

#### RSI (Relative Strength Index)
```
RSI 계산: 14일 기준
- RSI < 30: 과매도 (Bullish 반전 가능)
- RSI > 70: 과매수 (Bearish 반전 가능)
- RSI 40-60: 중립
```

#### 가격 모멘텀
```
1개월 수익률: 단기 모멘텀
3개월 수익률: 중기 모멘텀
6개월 수익률: 장기 모멘텀

3/3 양수: 강한 Bullish
3/3 음수: 강한 Bearish
```

### 변동성 분석

#### 볼린저 밴드
```
중심선: 20일 SMA
상단밴드: 20일 SMA + 2σ
하단밴드: 20일 SMA - 2σ

가격 < 하단밴드: 과매도 (평균회귀 예상)
가격 > 상단밴드: 과매수 (조정 예상)
```

#### ATR (Average True Range)
```
ATR: 14일 평균 변동폭
변동성 = ATR / 현재가

낮은 변동성 + 상승 추세: 안정적 상승
높은 변동성 + 하락 추세: 위험
```

## 성장 분석

### 매출 성장 품질

```
성장 품질 체크리스트:
1. 성장 일관성: 5년 연속 성장?
2. 성장 가속: 최근 성장률 > 과거 성장률?
3. 시장 점유율: 성장이 시장 확대 or 점유율 확대?
4. 가격 vs 볼륨: 가격 인상 or 판매량 증가?
```

### 마진 확대

```
Operating Leverage:
= 영업이익 성장률 / 매출 성장률

> 1.0: 레버리지 효과 (긍정적)
= 1.0: 비례 성장
< 1.0: 마진 압박
```

### 수익 품질

```
수익 품질 = FCF / Net Income

> 100%: 고품질 (현금 창출 우수)
80-100%: 양호
< 80%: 경계 (운전자본 또는 CapEx 부담)
```

## 센티먼트 분석

### 내부자 거래 신호

```
강한 Bullish:
- 다수의 경영진 매수
- 대규모 매수 (연봉의 10% 이상)
- CEO/CFO 직접 매수

Bearish 경고:
- 다수의 내부자 매도 (특히 스톡옵션 외)
- 연속적인 매도
- 주요 경영진 매도
```

### 뉴스 심리 가중치

```
뉴스 카테고리별 가중치:
- 실적 발표: 1.5x
- 경영진 변경: 1.3x
- 제품 출시: 1.2x
- 일반 뉴스: 1.0x
- 루머/추측: 0.5x
```

## 밸류에이션 프레임워크

### DCF 기반 내재가치

```
1. FCFF 추정
   FCFF = EBIT(1-T) + D&A - CapEx - ΔNWC

2. 성장률 추정
   - 1-5년: 애널리스트 컨센서스 또는 과거 CAGR
   - 6-10년: 점진적 감소
   - 영구: GDP 성장률 (~2.5%)

3. WACC 계산
   WACC = (E/V)×Re + (D/V)×Rd×(1-T)
   Re = Rf + β×(Rm-Rf)

4. 내재가치
   Enterprise Value = Σ(FCFF/(1+WACC)^t) + TV/(1+WACC)^n
   Equity Value = EV - Net Debt
```

### 상대 밸류에이션

```
동종 업계 비교:
- P/E 배수 비교
- EV/EBITDA 비교
- P/S 비교 (고성장 기업)

할인/프리미엄 정당화:
- 성장률 차이
- 수익성 차이
- 리스크 차이
```

### 안전마진 계산

```
안전마진 = (내재가치 - 현재가) / 내재가치

투자 결정 기준:
> 30%: 매력적 매수 기회
15-30%: 적정 매수
0-15%: 관망
< 0%: 고평가 (매도 고려)
```

## 종합 점수 계산

### 영역별 가중치

| 영역 | 가중치 | 근거 |
|------|--------|------|
| 펀더멘털 | 30% | 장기 가치의 근본 |
| 밸류에이션 | 25% | 진입 가격 중요성 |
| 성장 | 20% | 미래 가치 |
| 기술적 | 15% | 타이밍 최적화 |
| 센티먼트 | 10% | 단기 모멘텀 |

### 최종 신호 결정

```
종합 점수 = Σ(영역 점수 × 가중치)

점수 범위 (0-10):
- ≥ 7.5: Strong Buy
- 6.0-7.5: Buy
- 4.5-6.0: Hold
- 3.0-4.5: Sell
- < 3.0: Strong Sell
```
